# WIT Radio Music Library Seeding

## Goal

Start with a stable hybrid music library before depending fully on AI-generated tracks.

## Recommended Sources

- Pixabay Music
- Streambeats
- Free Music Archive
- self-produced tracks
- AI-generated bumpers and transitions

## Suggested Folder Layout

```text
/music/
  morning_acoustic/
  day_shift/
  dance_party/
  night_shift/
```

## Recommendation

Seed each show with:

- 50–100 real tracks initially
- 10–20 AI-generated tracks
- AI-generated transitions and station IDs

## Benefits

- more stable runtime
- less GPU usage
- better audio consistency
- easier testing
- fewer dead-air risks

## Long-Term Goal

Gradually replace larger portions of the station with AI-generated music as generation quality and automation improve.
