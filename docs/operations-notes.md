# WIT Radio Operations Notes

## Operational Priorities

1. Prevent dead air
2. Maintain stable audio levels
3. Keep transitions smooth
4. Avoid repetitive AI voice segments
5. Keep station IDs short

## Monitoring

Recommended checks:

- stream online status
- Icecast uptime
- disk usage
- CPU temperature
- ffmpeg process health
- AI generation queue size

## Recommended Runtime Habits

- restart generators during low-listener hours
- keep backups of playlists
- maintain fallback audio loops
- archive successful generated tracks
- rotate prompts regularly

## Suggested Future Features

- automatic silence detection
- fallback emergency playlist
- live DJ override mode
- auto social media clipping
- AI-generated album covers
- listener analytics
